18 # There's a saying that debugging code is about twice as hard as writing it in
19 # the first place. So if you're as clever as you can be when writing code, how
20 # will you ever debug it?
21
22 # AD faces a similar dilemma: if you write code that's as clever as the compiler
23 # can handle, how will you ever differentiate it? Differentiating makes clever
24 # code that bit more complex and the compiler gives up, usually resulting in
25 # 100x worse performance.
26
27 # Base's broadcasting is very cleverly written, and this makes differentiating
28 # it... somewhat tricky.
