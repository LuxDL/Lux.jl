8 """
9 BatchNorm(chs::Integer, λ=identity; initβ=zeros32, initγ=ones32,
10 affine = true, track_stats = true, ϵ=1f-5, momentum= 0.1f0)
11
12 [Batch Normalization](https://arxiv.org/abs/1502.03167) layer.
13
14 `BatchNorm` computes the mean and variance for each `D_1×...×D_{N-2}×1×D_N` input slice and normalises the input accordingly.
15
16 # Arguments
17
18 * `chs` should be the size of the channel dimension in your data (see below). Given an array with `N` dimensions, call the `N-1`th the channel dimension. For a batch of feature vectors this is just the data dimension, for `WHCN` images it's the usual channel dimension.
19 * After normalisation, elementwise activation `λ` is applied.
20 * If `affine=true`, it also applies a shift and a rescale to the input through to learnable per-channel bias β and scale γ parameters.
21 * If `track_stats=true`, accumulates mean and var statistics in training phase that will be used to renormalize the input in test phase.
22
23 Use [`testmode`](@ref) during inference.
24
25 # Examples
26
27 ```julia
28 m = EFL.Chain(
29 EFL.Dense(784 => 64),
30 EFL.BatchNorm(64, relu),
31 EFL.Dense(64 => 10),
32 EFL.BatchNorm(10)
33 )
34 ```
35 """

243 """
244 WeightNorm(layer::AbstractExplicitLayer, which_params::NTuple{N,Symbol}, dims::Union{Tuple,Nothing}=nothing)
245
246 Applies [weight normalization](https://arxiv.org/abs/1602.07868) to a parameter in the given layer.
247
248 ``w = g\\frac{v}{\\|v\\|}``
249
250 Weight normalization is a reparameterization that decouples the magnitude of a weight tensor from its direction. This updates the parameters in `which_params` (e.g. `weight`) using two parameters: one specifying the magnitude (e.g. `weight_g`) and one specifying the direction (e.g. `weight_v`).
251
252 By default, a norm over the entire array is computed. Pass `dims` to modify the dimension.
253 """
254 struct WeightNorm{which_params,L<:AbstractExplicitLayer,D} <: AbstractExplicitLayer
255 layer::L
256 dims::D
257 end

261 ) where {N}

263 end

265 function initialparameters(rng::AbstractRNG, wn::WeightNorm{Val{which_params}}) where {which_params}

280 end
